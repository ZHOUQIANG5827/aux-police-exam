## 功能特性

- **套题模考**（最醒目主按钮）：按批次聚合抽卷，模拟真实考场，交卷后统一判分、展开答案与解析。
- **定向刷题**：按批次 / 题型筛选，**点击即判**（单选 / 判断点选项即判对错并展示解析，多选提交后判分），无「先选后核对」的多余步骤。
- **随机抽题**：随机出题，点击即判。
- **只看未掌握**：标记错题，专攻薄弱点。
- **面试模块**（深圳）：结构化真题按板块刷 + 录音演练 + 语音转文字预览。
- **判断题**：选项统一显示为 **A. 正确 / B. 错误**，答案区明确显示「正确 / 错误」，无歧义。
- **性能**：首屏懒加载、无内联大图，HTML 轻量、秒开。

## 部署

Cloudflare Pages **连接本 GitHub 仓库，`git push` 到 `main` 自动重新部署**。

- 每个城市站可各自建独立 Pages 项目 + 独立域名（构建输出目录设为对应子目录，如 `sz` / `hz`）。
- 门户站输出目录为根目录 `.`。
- 详细步骤见 [`DEPLOY-STEPS.md`](DEPLOY-STEPS.md)；新增城市参考 [`DEPLOY-HUIZHOU.md`](DEPLOY-HUIZHOU.md)。

## 本地预览

直接双击打开对应的 `index.html` 即可（面试录音需麦克风权限，`file://` 打开会受限，建议用本地 HTTP 服务或线上环境）。

## 技术说明

- 站点由 `rcj-exam-builder` 技能模板 `viewer.html` 生成。
- 单文件自包含：题库数据、配置、样式、脚本全部内联，0 外部 `script`/`src` 引用。
- 不提供站点内「导出题目 / 打印」按钮（对刷题价值低、且稀释付费完整版卖点）。
