# 深圳辅警真题卡组

深圳辅警**面试 + 笔试**历年真题卡组，含批次 / 题型 / 标签筛选、套题模考、定向刷题点击即判、随机抽题、只看未掌握，面试模块支持录音演练与语音转文字预览。

> 属 [RCJ 辅警题库](../README.md) 平台的深圳站。笔试已结束，当前默认优先展示**面试**。

## 页面说明

- `index.html`：深圳站主入口，顶部 tab 切换 **面试 / 笔试**，完全自包含单文件。
  - **面试**：结构化真题按板块刷 + 录音演练 + 语音转文字预览。
  - **笔试**：单选 / 多选 / 判断，点选即判分并展示解析，套题模考按批次抽卷。
- `written.html`：早期分离的笔试单页（已并入 `index.html`，保留作备份）。

## 本地预览

直接双击打开 `index.html` 即可（麦克风录音需 HTTPS 环境，本地 `file://` 打开会受限）。

## 部署

已连接 Cloudflare Pages，推送到 `main` 分支自动触发发布（构建输出目录设为 `sz`）。
