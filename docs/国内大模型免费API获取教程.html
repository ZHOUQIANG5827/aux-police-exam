📖 国内大模型免费 API 获取教程

配套《深圳辅警面试真题卡组 · 离线版》的 AI 点评 / 语音转写功能

⚠️ 本刷题站的「AI 点评」「语音转写」需要你自己的大模型 API Key。 Key 只存在你本机浏览器（localStorage），不会上传给任何人，请放心填写。 本教程只讲「怎么免费拿 Key」，各平台免费额度会随政策变动，具体以官网为准。

一、先分清两种用途

本站的接口分两块，填的 Key 可以相同也可以不同：

  • ☁️ 语音转写（ASR）：把你的录音变成文字。需要支持「语音识别」的接口，目前最稳的是 硅基流动 SenseVoice（模型 FunAudioLLM/SenseVoiceSmall）。 其它纯对话大模型（通义/智谱/DeepSeek 等）没有语音识别接口，不能用于转写。
  • 🤖 AI 点评：把「题目 + 你的口述文字」发给大模型打分。只要是 OpenAI 兼容的对话接口都能用， 硅基流动 / 通义千问 / 智谱 GLM / DeepSeek / 月之暗面 / 腾讯混元 / 火山豆包 等都可。
💡 省事方案：直接去 硅基流动（siliconflow.cn） 申请一个 Key，转写和点评都用它， 模型分别填 FunAudioLLM/SenseVoiceSmall 和 deepseek-ai/DeepSeek-V3（或 Qwen/Qwen2.5-7B-Instruct 等）。一个 Key 全搞定。

三、通用拿 Key 步骤（每个平台都差不多）

  1. 打开上面的官网，用手机号注册 / 登录。
  2. 在左侧菜单找「API 密钥 / API Key / 访问控制 / 令牌 / 鉴权」。
  3. 点「创建密钥 / 新建 Key」，生成后立刻复制保存（通常只显示一次，形如 sk-xxxx）。
  4. 查看「免费额度 / 体验额度 / 资源包」：新用户一般送几元到几十元，或某些模型直接免费。
  5. 回到刷题站 ⚙️ 语音设置，把 Key + 模型名 + Base URL 填进去，点「保存」。

四、在本刷题站怎么填

① 语音转写（录音→文字）

  • 转写引擎选 ☁️ 云端 API
  • API Base URL：https://api.siliconflow.cn/v1
  • API Key：你的硅基流动 Key
  • 模型名：FunAudioLLM/SenseVoiceSmall

② AI 点评（文字→打分）

  • 先勾选「✅ 启用 AI 点评」
  • API Base URL / Key / 模型名：填你选的平台（硅基流动可复用上面的 Key）
  • 模型名示例：deepseek-ai/DeepSeek-V3 / qwen-plus / glm-4-flash
💡 填完点「🧪 测试连接」验证。转写完题目，再点「🤖 AI 点评」即可出分。

五、常见问题

  • 提示 401 / Key 无效：多半是复制时带进了空格，或 Key 已过期 / 额度用完。重新复制完整 Key 再试。
  • 手机打不开 / 转写没反应：推荐用☁️云端 API（国内直连），比浏览器内置/Whisper 离线兼容性更好；华为/微信内置浏览器也支持。
  • 只想点评、不录音：可以手动在「自定义题」里贴文字，再点 AI 点评，不一定要录。。
  • 配置不生效：更新题库后请按 Ctrl+F5（Mac：Cmd+Shift+R）硬刷新。
🔒 安全提醒：Key 仅存你自己的浏览器，不要截图发给别人、不要填到不明网站。 如担心泄露，去平台「删除 / 重置密钥」即可，旧 Key 立刻失效。
